# Elevator-controller-using-FSM.-
This project implements an elevator controller where using FSM, it gets and fulfills the request got from users. if any emergency situation like overload, power cut, door obstruction, system will enter emergency state. 
#features
- 15 floors building elevator controller, can be adjust to any no of floors.
- emergency state for emergency situation.
#tools used
- Vivado
#Results
-current floor, current FSM state
- pending request
#how to run
- compile using vivado
- run testbench
- view waveform/transcript
#output
- transcript
#learnings
- use of FSM to design any system.
